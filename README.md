
# Music Recommendation System

## Overview

This project is a Music Recommendation System that suggests songs based on user preferences using collaborative filtering techniques. It integrates with the Spotify API to fetch music data and provide personalized recommendations.

The objective of this project is to enhance user experience by delivering relevant and accurate music suggestions.

---

## Features

* Personalized music recommendations
* Search and explore songs
* Collaborative filtering-based recommendation engine
* Integration with Spotify API
* Efficient and responsive system

---

## How It Works

1. Collects user preferences or selected songs
2. Applies collaborative filtering to identify similar users or items
3. Computes similarity scores
4. Recommends top matching songs

---

## Installation

1. Clone the repository

```
git clone https://github.com/Dharsh-04/Music_Recommendation_System.git
cd Music_Recommendation_System
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the application

```
python app.py
```
